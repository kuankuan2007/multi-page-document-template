# 多页文档模板

[English README](./README.md) | 中文

---

## 简介

这是一个“项目文档多页化”的前端模版：将长文档拆成多篇 Markdown，并以目录 + 正文的形式在浏览器中展示。

## 文档

教程、配置说明与更多细节请看： [Example](https://multi-page-document-template.pages.kuankuan.site/)

## 启动开发服务器（最小步骤）

本项目在 Node `20.11.0` 上开发（见 `.nvmrc`）。

```bash
npm i
npm run dev
```
